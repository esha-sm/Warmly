# Warmly 🧊

AI-powered cold outreach tool that writes and sends hyper-personalized emails from enriched lead data.

## Stack
- Next.js 16 + Tailwind
- Claude API (claude-sonnet-4-6)
- Clay for lead enrichment
- Instantly for sending + tracking
- Vercel for deployment

## How it works
1. Upload a CSV from Apollo, Clay, or LinkedIn
2. Paste your background
3. Claude writes a personalized email per lead using their company summary, GTM challenge, and recent news
4. Emails sync to Instantly for automated sending, follow-ups, and reply tracking
5. Reply data feeds back into Claude to improve future emails
